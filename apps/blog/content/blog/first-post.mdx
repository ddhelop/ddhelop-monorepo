# 첫 번째 블로그 포스트

안녕하세요! 이 글은 제 블로그의 첫 번째 포스트입니다. 이 블로그에서는 프론트엔드 개발, Next.js, React 등에 관한 내용을 다룰 예정입니다.

## 소개

Next.js와 MDX를 사용하여 개발한 이 블로그는 다음과 같은 특징이 있습니다:

- 정적 생성을 통한 빠른 로딩 속도
- MDX를 통한 마크다운 + React 컴포넌트 작성
- 코드 하이라이팅 지원
- 반응형 디자인

## 코드 예시

```jsx
// Next.js 컴포넌트 예시
import { useState } from 'react';

export default function Counter() {
  const [count, setCount] = useState(0);

  return (
    <div>
      <p>현재 카운트: {count}</p>
      <button onClick={() => setCount(count + 1)}>증가</button>
    </div>
  );
}
```

# Tailwind CSS v4 사용 방법

Tailwind CSS는 최근 V4 버전을 출시했습니다. 이번 글에서는 V4의 주요 변경사항과 사용 방법에 대해 알아보겠습니다.

## 주요 변경사항

Tailwind CSS v4에서는 다음과 같은 주요 변경점이 있습니다:

- 설정 파일(tailwind.config.js)이 더 이상 필수가 아님
- 새로운 컬러 시스템 도입 (OKLCH)
- 성능 향상과 번들 크기 최적화
- 익스텐션 API 개선

## 설치 방법

Tailwind CSS v4는 다음과 같이 설치할 수 있습니다:

```bash
npm install tailwindcss@latest
```

## 기본 설정

```css
/* globals.css */
@import 'tailwindcss';

@source inline("text-primary font-bold bg-gray-100");

@layer base {
  :root {
    --color-primary: oklch(0.6 0.12 270);
  }
}
```

## 예제 코드

```jsx
export default function Example() {
  return (
    <div className="p-4 bg-white rounded-lg shadow">
      <h2 className="text-xl text-primary font-bold mb-4">
        Tailwind CSS v4 예제
      </h2>
      <p className="text-gray-700">
        Tailwind CSS v4를 사용한 간단한 카드 컴포넌트입니다.
      </p>
      <button className="mt-4 px-4 py-2 bg-primary text-white rounded hover:opacity-80 transition-opacity">
        자세히 보기
      </button>
    </div>
  );
}
```

## 결론

Tailwind CSS v4는 더 간결해진 설정과 향상된 성능으로 프론트엔드 개발 경험을 한층 더 개선했습니다. 이전 버전에서 마이그레이션하는 경우 약간의 변경이 필요할 수 있지만, 전반적으로 더 나은 개발 경험을 제공합니다.
